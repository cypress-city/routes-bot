# players-page
A Discord bot with information about routes (also known as "intermissions") in Mario Kart World.

Add the official bot to your server: https://discord.com/oauth2/authorize?client_id=1469567980626116740

## Setup
1. Create a `.env` file in the main directory with the token for your Discord application:
```
TOKEN=[your token]
```
2. Install the necessary dependencies via command line:
```commandline
python -m pip install -r requirements.txt
```

## Operation
```commandline
python main.py
```
